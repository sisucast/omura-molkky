# Claude Code 引き継ぎ用エクスポート

このフォルダは、第2回大村モルックカップ LP の最終調整を Claude Code で行うための引き継ぎ用です。

## 同梱ファイル
- `index.html` : 現時点の最終版HTML
- `README.md` : 引き継ぎメモ

## 現在の反映済み内容
- LPデザインは、落ち着きすぎないポップ寄りの方向で再構築済み
- スマホ表示でもヘッダーカテゴリは表示
- フローティングCTAを常時表示
- フローティングCTAの文言は `ENTRY`
- ヘッダー内の申込みボタンは削除済み
- フローティングCTA以外の申込みボタン文言は `参加申し込みはこちら`
- 開催時間は `9時～13時`
- 個人申込み文言は以下に統一済み
  - `チームメンバーが集まらない場合、個人での申し込みも可能です。その場合、主催側で当日マッチングします。`
- ヒーロー文言は以下に変更済み
  - `投げて、笑って、盛り上がる。第2回大村モルックカップを開催します。`

## Claude Code での推奨最終仕上げポイント
1. 余白・文字サイズの微調整
2. スマホ時のフローティングCTAの高さと重なり確認
3. 画像パスの最終確認
4. Lighthouse観点での軽微な調整
5. セクション間の視線誘導の微修正

## 注意点
- HTML内で参照している画像パス（例: `uploads/...`）が実環境に存在するかは別途確認が必要です。
- フローティングCTAが固定表示のため、ページ最下部で他要素と重なっていないか最終確認を推奨します。
